COVID-19


This dataset contains information about the COVID-19 pandemic, it has a few fields the relevant are how many sick, dead, or recovered were in a specific date and a specific country. this dataset was created by the Kaggle user "Devakumar kp" from India, he uses a few sources to get to one data set with more country and the most reliable data (his code).

The dataset

field name explanation
Country/Region Name of a country/region.
Date The date of the report.
Confirmed The number of cases of people confirmed to have COVID-19.
Deaths The number of cases of people dead by COVID-19.
Recovered The number of cases of people who recovered from COVID-19.
